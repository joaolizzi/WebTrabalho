DOM e Interatividade

Seleção, Alteração, Eventos e Manipulação de Elementos

O DOM (Document Object Model) é uma interface que representa documentos HTML de forma estruturada, permitindo que scripts possam acessar, modificar e interagir com os elementos da página. A manipulação do DOM é fundamental para criar páginas web dinâmicas e interativas, tornando a experiência do usuário muito mais rica.

1. Seleção de Elementos

O primeiro passo para manipular o DOM é selecionar os elementos que queremos alterar ou interagir. Existem diversos métodos para isso, sendo os mais comuns getElementById e querySelector.

  • getElementById: Seleciona um elemento pelo seu atributo id. Retorna apenas um elemento.
  • querySelector: Seleciona o primeiro elemento que corresponde a um seletor CSS (pode ser .classe, #id, tag, etc).

Após selecionar, podemos alterar propriedades, estilos ou adicionar eventos. Veja o exemplo abaixo, onde mudamos a cor do título e do subtítulo:

// Usando getElementById let titulo = document.getElementById('titulo'); titulo.style.color = 'blue'; // Usando querySelector let subtitulo = document.querySelector('.subtitulo'); subtitulo.style.color = 'green';

2. Alterar Conteúdo e Estilos

Depois de selecionar um elemento, podemos modificar seu conteúdo e aparência. Os métodos mais usados são:

  • innerText: Altera apenas o texto do elemento, ignorando qualquer HTML.
  • innerHTML: Permite inserir HTML dentro do elemento, possibilitando formatações.
  • style: Permite alterar estilos CSS diretamente pelo JavaScript.

No exemplo abaixo, mudamos o texto de um parágrafo, inserimos HTML em outro e alteramos a cor de fundo de uma caixa:

// innerText document.getElementById('texto').innerText = "Texto alterado!"; // innerHTML document.getElementById('html').innerHTML = "<strong>HTML alterado com negrito</strong>"; // style document.getElementById('caixa').style.backgroundColor = "lightblue";

3. Eventos (click, keyup, submit)

Eventos são ações que ocorrem na página, como cliques, digitação ou envio de formulários. Podemos "ouvir" esses eventos e executar funções quando eles acontecem, tornando a página interativa.

  • click: Disparado quando o usuário clica em um elemento.
  • keyup: Disparado quando o usuário solta uma tecla após digitar em um campo.
  • submit: Disparado ao enviar um formulário.

Veja exemplos práticos de cada evento:

Evento Click

Ao clicar no botão, um alerta é exibido na tela.

document.getElementById('btnClick').addEventListener('click', function() { alert("Você clicou no botão!"); });

Evento Keyup

Ao digitar no campo, o texto digitado aparece logo abaixo em tempo real.

document.getElementById('inputKeyup').addEventListener('keyup', function(e) { document.getElementById('resultadoKeyup').innerText = "Você digitou: " + e.target.value; });

Evento Submit

Ao enviar o formulário, a página não recarrega e um alerta confirma o envio.

document.getElementById('formulario').addEventListener('submit', function(e) { e.preventDefault(); alert("Formulário enviado com sucesso!"); });

O uso de eventos é fundamental para criar páginas responsivas, que reagem às ações do usuário de forma instantânea.

4. Criar e Remover Elementos

Além de alterar elementos existentes, também podemos criar novos elementos dinamicamente e adicioná-los à página, ou remover elementos quando necessário. Isso é muito útil para listas, tabelas, cards e outros conteúdos dinâmicos.

  • createElement: Cria um novo elemento HTML via JavaScript.
  • appendChild: Adiciona um elemento como filho de outro elemento.
  • removeChild: Remove um elemento filho.

No exemplo abaixo, ao clicar no botão, um novo item é adicionado à lista:

document.getElementById('btnCriar').addEventListener('click', function() { let novoItem = document.createElement('li'); novoItem.innerText = "Novo item"; document.getElementById('lista').appendChild(novoItem); });

    Com essas técnicas, é possível criar aplicações web dinâmicas, como listas de tarefas, chats, sistemas de comentários e muito mais, tudo sem recarregar a página.

    Resumo

    Manipular o DOM é uma das habilidades mais importantes para quem desenvolve para web. Com a seleção de elementos, alteração de conteúdo e estilos, uso de eventos e criação dinâmica de elementos, é possível transformar páginas estáticas em experiências ricas e interativas para o usuário.

    Pratique bastante esses conceitos e experimente criar suas próprias interações para fixar o aprendizado!
